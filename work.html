Weather App

The weather application is a solution built using HTML, CSS and Javascript. The application provides the weather information of different cities anywhere in the world. The application also provides the maximum and minimum temperature 5 days forecast of the selected city. The weather data was obtained through and API integration to weather data API powered by openweathermap.org

Naija Jollof

Nigerian JollofRice is an everyday dish that is prepared using tomatoe paste, rice and local seasoning. Nigerians belive their version of this dish is the best in West Africa, although it is made in other West African countries and is belived to have its origin from Senegal. This dish is now accepted internationally and one of the favourite cuisines of foreign nationals when they visit Nigeria. This web page was built with HTML and CSS and the link below will lead you to one of my favourite Jollof Huts to Order a Naija Jollof dish for yourself.
